PASSI PER POTER ESEGUIRE IL CODICE :
- Necessario avere gradle (eventualmente installarlo)
- Una volta aperto il gradle project nell'ide(riferimento : Eclipse), selezionare il progetto
- Eseguire(da eclipse, gradle task bar) l'init e la build del gradle project.
- Infine eseguire il codice con run (sempre dalla gradle task bar)
NECESSARIO : 
- Stabilire una connessione con mySqlServer sulla port 3006(utilizzato xampp)
- il DDL è già incluso nel progetto, si può eventualmente anche trovare al link https://github.com/AlePioggia/DDL-BDD
